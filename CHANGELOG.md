# BrochureCraft / 画册模板 更新日志

## 近期功能更新

### 一、BrochureCraft 编辑器功能

1. **内页模式增删编辑**
   - 内页模式下支持新增/删除岗位、单元、知识点。
   - 新增知识点自动分布在单元圆形四周，避免与现有知识点重叠。
   - 删除单元后，后续单元自动左移补位，并同步删除对应时间轴连接线。
   - 单元顺序统一为从左到右 1–7 顺序填充。

2. **预览区域缩放**
   - 中间预览区新增缩放控制条，支持放大、缩小、适应窗口，方便查看细节。
   - 修复缩放模式下无法向左滚动、无法查看全部 HTML 内容的问题。

3. **压缩包导入支持**
   - 支持上传 `.zip` 压缩包并自动解压读取 HTML。
   - 支持一个压缩包中同时包含封面（`cover.html`）和内页（`inner.html`）。
   - 导入后可在左上角切换「封面 / 内页」编辑。
   - 同时保存封面、内页两个版本的 HTML（共 4 个文件）。
   - 修复组合导入时封面识别为空的问题。

4. **文本编辑修复**
   - 修复内页模式下节点标题被错误拆分成多个编辑框的问题。
   - HTML 字段保存时自动将换行转换为 `<br>`。

5. **界面与交互优化**
   - 移除「导出 HTML」按钮。
   - 右侧「AI 内容助手」更名为「AI 编辑助手」。
   - 页面结构编辑区域默认折叠，可点击展开/收起。
   - 操作按钮符号统一改为中文「新增」「删除」。
   - 🗑「清理上下文」按钮仅在有对话内容时显示。
   - 💡 AI 参考案例替换为更贴合画册场景的营销/编辑类示例。
   - 全面美化 BrochureCraft 界面样式。

### 二、模板优化

1. **template1.html（原 brochure_template.html / demo_clean.html）**
   - 清理 HTML 结构，修复全页白色遮罩问题。
   - 移除 Pencil 残留标记，修正路径与 SVG 属性。
   - p2「人才培养方案」页面右上角 logo 添加 `header_logo` 可替换图片字段。
   - p2 页面顶部「知与」logo 改为可替换图片字段 `p2_zhiyu_logo`，默认使用 `logo.png`。

2. **template2.html（原 demo_clean.html）**
   - 清理模板结构，移除多余样式与脚本残留。
   - 优化时间轴样式，提升 `html2canvas` 导出兼容性。
   - 调整知识点标签位置，使其分布在单元圆形四周。
   - 收紧单元圆形内标题行间距（`line-height` 从 38px 调整为 30px），让两行标题更紧凑。

3. **模板规范**
   - 更新 `模板制作规范.md`，补充内页模板、组合导入、AI 编辑约束等内容。
   - 基于规范调整 AI system prompt，增加模板规范与背景色修改约束。
