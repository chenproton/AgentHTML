# BrochureCraft 使用手册

> 知与企业画册在线制作系统 v2.16
> 适用于第一次使用 BrochureCraft 的新人快速上手。

---

## 一、系统简介

BrochureCraft 是一个基于浏览器的画册在线制作工具，支持：

- 导入 HTML 模板并可视化编辑
- AI 辅助修改文案、样式、图片
- 导出高清图片或 PDF
- 同时管理「封面」与「内页」两种画册结构

所有数据（模板、编辑进度、AI 对话）默认保存在当前浏览器本地缓存中，不同浏览器/设备互不干扰。

---

## 二、界面总览

打开 `BrochureCraft.html` 后，页面分为四个区域：

```
┌─────────────────────────────────────────────────────────────┐
│  顶部工具栏                                                    │
├──────────┬──────────────────────────────┬───────────────────┤
│          │                              │                   │
│ 左侧     │      中间预览区               │   右侧编辑区       │
│ 页面列表  │      （A4 / 长图画布）         │   （结构 + AI）    │
│          │                              │                   │
└──────────┴──────────────────────────────┴───────────────────┘
```

### 1. 顶部工具栏

| 按钮 | 作用 |
|------|------|
| ⚙️ AI 配置 | 设置 AI 接口地址、API Key、模型、温度参数 |
| 📂 导入模板 | 导入官方生成的画册模板压缩包 |
| 🗑 清空缓存 | 清空当前浏览器所有模板、编辑记录、AI 对话 |
| 📥 导出图片 | 将当前页面导出为 PNG 高清图片 |
| 📄 导出 PDF ▾ | 导出标准版（压缩）或印刷版（无压缩）PDF |

> 导入模板后，如果同时包含封面+内页，工具栏标题旁会出现「封面 / 内页」切换按钮。

### 2. 左侧页面列表

- 显示当前导入的所有页面缩略图与名称。
- 点击任意页面即可在中间预览区切换显示。
- 绿色小点表示该页面已有内容字段被填充。

### 3. 中间预览区

- 实时展示当前选中的页面效果。
- 可以直接点击页面上的文字或图片进行编辑。
- 底部有缩放控制条：放大、缩小、适应窗口，方便查看细节。

### 4. 右侧编辑区

分为上下两部分：

- **页面结构面板**：仅在内页模式下出现，用于管理岗位、单元、知识点。
- **AI 编辑助手**：与 AI 对话，描述修改需求，AI 会自动生成并应用代码。

---

## 三、导入模板

点击顶部「📂 导入模板」，选择官方生成的 `.zip` 模板压缩包即可开始制作。

### 压缩包内容说明

官方模板压缩包需同时包含封面与内页：

- `cover`：封面部分
- `inner`：内页部分

导入后，系统会自动识别封面与内页，并在左上角显示「封面 / 内页」切换按钮，方便随时切换编辑。

---

## 四、基础编辑

### 1. 点击页面直接编辑

在预览区：

- **点击文字**：弹出文本编辑框，修改后保存即可。
- **点击图片**：弹出图片上传框，可替换为新图片。

> 所有修改会实时反映在预览区，并自动保存到浏览器缓存。

### 2. 预览区缩放

预览区底部固定悬浮缩放条：

| 按钮 | 作用 |
|------|------|
| − | 缩小 |
| 100% 等 | 当前缩放比例，点击可快速重置为「适应窗口」 |
| + | 放大 |

缩放后可通过滚动条查看画布任意位置。

---

## 五、内页结构编辑

导入内页模板后，右侧「页面结构」面板会显示当前岗位与单元层级：

```
岗位 1：智能仓储运营
  └─ 单元 1：智能仓储系统运营
      ├─ 知识点 1
      ├─ 知识点 2
      └─ 知识点 3
  └─ 单元 2：物流数据采集可视化
      ├─ 知识点 1
      └─ 知识点 2
```

### 操作说明

| 操作 | 方法 |
|------|------|
| 展开/折叠面板 | 点击面板右上角「展开」/「折叠」按钮 |
| 新增岗位 | 点击面板右上角「新增岗位」按钮 |
| 新增单元 |  hover 岗位，点击「新增」按钮 |
| 新增知识点 | hover 单元，点击「新增」按钮 |
| 删除 | hover 任意层级，点击「删除」按钮 |

### 注意事项

- 删除岗位会删除该岗位下所有单元和知识点。
- 删除单元后，后续单元会自动左移补位，保持 1–7 的顺序。
- 删除单元会同步删除对应时间轴连接线。
- 新增知识点会自动分布在单元圆形四周，不会与已有知识点重叠。

## 六、AI 编辑助手

### 1. 配置 AI

首次使用 AI 前，点击顶部「⚙️ AI 配置」，填写：

- **API Base URL**：OpenAI 兼容格式的接口地址
- **API Key**：你的 API Key
- **模型名称**：如 `deepseek-v4-flash`
- **内容发散性**：0~1 之间，建议 0.7

填写后点击「保存」即可。

### 2. 发送指令

在右下角输入框中描述你想修改的内容，例如：

- 「把页面底色改成浅蓝色」
- 「将产业认知描述精简到 100 字以内」
- 「把全文中的『物流』统一替换为『智慧物流』」
- 「让封面标题居中并加大两号」
- 「把这段岗位介绍写得更正式、专业」

点击「发送」后，AI 会返回修改说明并自动应用到页面上。

### 3. 参考案例

输入框上方有一排「💡 试试问这些」快捷案例，点击即可自动填入输入框。

### 4. 附件功能

点击输入框左侧「附件」按钮，可上传参考图片或文档，作为 AI 修改的参考。

### 5. 清理上下文

当对话变长或 AI 理解偏离时，点击 AI 头像旁的「🗑 清理上下文」按钮，仅清空当前页面的 AI 对话记录。

> 该按钮只在当前页面有过对话时才显示。

## 七、导出作品

编辑完成后，点击顶部导出按钮：

### 1. 导出图片

- 将当前页面渲染为 PNG 高清图片。
- 内页导出尺寸可在 AI 配置中设置（默认 9920×6732）。

### 2. 导出 PDF

- **标准版（压缩）**：文件较小，适合在线传阅。
- **印刷版（无压缩）**：画质最高，适合打印。
- 封面模板按 A4 多页导出；内页模板按单张大图导出。

## 八、多用户与并发说明

- **数据隔离**：BrochureCraft 所有数据保存在浏览器本地（`localStorage`），不同用户、不同浏览器之间互不干扰。
- **AI 并发**：AI 调用在各自浏览器端独立发起，服务器本身不维护用户状态。如果多人同时使用同一 API Key，请注意 API 提供方的并发限制。

## 九、常见问题

### Q1：导入后页面空白？

- 检查模板文件是否有效。
- 封面模板需包含 `.a4-page` 元素。
- 内页模板需包含大尺寸的 `.page` 元素。
- 请确保压缩包中同时包含封面与内页两个模板文件。

### Q2：AI 没有响应？

- 检查「⚙️ AI 配置」中的 API Base URL、API Key、模型名称是否填写正确。
- 打开浏览器开发者工具（F12）查看网络请求错误信息。

### Q3：图片替换后没变化？

- 图片字段依赖 `background-image` 或 `<img>` 标签，确保模板中正确标记了 `data-type="image"`。
- 尝试刷新页面后重新导入。

### Q4：不小心清空了缓存怎么办？

- 清空缓存后无法恢复，请重新导入模板。
- 建议重要作品及时导出图片/PDF 备份。

## 十、模板制作小贴士

- 文字字段使用 `data-field="字段名" data-type="text"`。
- HTML 富文字段使用 `data-type="html"`。
- 图片字段使用 `data-field="字段名" data-type="image"`，并通过 `background-image` 展示。
- 避免在模板中使用全屏白色遮罩，否则修改背景色时会被覆盖。

更多规范请参考：`模板制作规范.md`。
